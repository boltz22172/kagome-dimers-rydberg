# Data and code for "Engineering resonating kagome dimers in Rydberg-atom arrays"

This repository contains the data and scripts used to generate Fig. 29
in the paper:

X. Wang and E. J. Mueller,
"Engineering resonating kagome dimers in Rydberg-atom arrays",
Physical Review A (2025).

## Contents
- `data/`: numerical data used in Fig. 29
- `scripts/`: scripts to reproduce the figure

## Usage
Run the script in `scripts/` to reproduce Fig. 29.
